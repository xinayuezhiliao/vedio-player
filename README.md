# VIP视频解析工具

一款基于 tkinter 开发的桌面小工具，支持解析播放腾讯视频、优酷、爱奇艺等平台的 VIP 影视内容。

---

## 功能特性

- 支持解析腾讯视频（v.qq.com）、优酷（youku.com）、爱奇艺（iqiyi.com）等主流平台
- 多接口自动切换：默认接口 + 两个备用接口，解析失败时可换接口重试
- 播放历史自动记录（最近 50 条），并支持手动指定影片名称
- 收藏库功能：常用影片一键收藏，随时打开直接播放
- 自动识别视频标题，也支持手动填写影片名称
- 数据本地保存，换电脑重装只要把 `video_parse_history.json` 和 `video_favorite.json` 复制到同目录即可

---

## 安装运行

### 环境依赖

- Python 3.7+
- 依赖库：`requests`（首次运行会自动提示安装）

### 运行方式

```bash
python video_player.py
```

首次运行若提示缺少 `requests`，程序会自动调用 pip 安装。

### 打包为 exe（可选）

使用 PyInstaller 打包后，`history.json` 和 `favorite.json` 会保存在 exe 同目录下，迁移方便。

## 界面介绍

### 主界面布局

```
+--------------------------------------------------+
|              VIP视频解析工具                       |
+--------------------------------------------------+
|  目标网址：  [________________________] [清空]    |
|  影片名称：  [________________________]          |
|  解析接口：  [默认接口        ▼]                  |
|             [添加到收藏]  [我的收藏库]            |
|                 [播放视频]                        |
+--------------------------------------------------+
|  历史解析记录                    |  播放选中      |
|  - 视频标题1                     |  删除选中      |
|  - 视频标题2                     |  清空所有      |
|  ...                            |               |
+--------------------------------------------------+
```

- **目标网址**：粘贴视频播放页 URL，如 `https://v.qq.com/x/cover/xxx.html`
- **影片名称**：可不填，程序会自动从 URL 或页面抓取标题
- **解析接口**：默认优先，遇到解析失败可换备用接口
- **历史记录**：左侧列表，选中后点击"播放选中"按钮播放

### 收藏库窗口

点击"我的收藏库"打开，支持：

- 播放选中影片
- 删除单条收藏
- 清空全部收藏

---

## 使用步骤

1. 复制目标视频的播放页面 URL
2. 粘贴到"目标网址"输入框
3. （可选）手动填写影片名称，方便识别
4. 选择解析接口（默认接口一般可用）
5. 点击"播放视频"，浏览器将自动打开并播放

> 提示：解析依赖第三方接口，若接口失效请切换备用接口，或等待接口恢复。

---

## 数据存储

运行后会生成两个 JSON 文件，存放在程序或脚本同目录下：

| 文件 | 内容 |
|------|------|
| `video_parse_history.json` | 播放历史记录 |
| `video_favorite.json` | 收藏影片列表 |

迁移或重装时，保留这两个文件即可恢复历史和收藏。

---

## 注意事项

- 本工具仅供学习交流，请勿用于商业用途，尊重影视版权
- 解析接口来自第三方，不保证长期可用，发现失效请反馈或等待更新
- 程序会定期自动更新历史记录中的视频标题（如"待更新_xxx"会逐步替换为真实标题）
