# VIP视频解析工具

一款基于 tkinter 的桌面小工具,支持解析播放腾讯/优酷/爱奇艺/芒果TV 等平台的视频资源。

## 功能特性

- ✅ **本地资源索引库** — 自己手动维护,搜什么有什么(真正能搜到的搜索)
- ✅ **收藏库** — 一键收藏常用影片,带"导出到本地库"按钮
- ✅ **历史记录** — 自动记录最近 50 条解析记录,3 行/条带时间和接口
- ✅ **播放链** — 自动探测 VLC → Chrome → 系统浏览器,任一可用就用
- ✅ **解析接口** — 默认接口 + 2 个备用,接口失效可换
- ✅ **跳转平台搜索页** — 本地搜不到时一键跳到 4 大平台搜索页,用户手动复制 URL
- ✅ **零外部依赖** — 用 Python 标准库 urllib,无需 pip install
- ⚠️ **诚实声明**:直接爬取爱奇艺/腾讯/优酷/芒果的搜索结果是 JS 渲染的 SPA,
  纯 HTTP 抓不到。需要登录态/反爬绕过的"真搜索"不在本工具范围内,
  改用"本地索引库 + 跳转搜索页"的替代方案。

## 安装运行

### 环境要求
- Python 3.7+(标准库即可,**无第三方依赖**)

### 运行
```bash
python video_player.py
```

### 打包为 exe(可选)
```bash
pyinstaller --onefile --windowed --icon=movie_icon.ico video_player.py
```

## 使用流程

### 方式 A:本地资源库(推荐,真能搜到)
1. 先去爱奇艺/腾讯/优酷/芒果搜你想看的影片,复制 URL
2. 在主界面点 `+ 本地库` 把 URL 加进本地资源库
3. 下次在搜索框输影片名,本地库里有就能搜到

### 方式 B:本地搜不到时跳浏览器
1. 在搜索框输关键词,本地无结果
2. 点 `🌐 浏览器搜`,弹窗选平台,跳到平台搜索页
3. 在浏览器里搜,复制真实 URL,回到主界面粘贴播放

### 方式 C:手动播放
1. 直接在"目标网址"粘贴 URL
2. 点 `▶ 播放视频`,自动用播放链(VLC → Chrome → 浏览器)打开

## 数据存储

所有数据存程序同目录:

| 文件 | 内容 |
|------|------|
| `video_parse_history.json` | 播放历史(最近 50 条) |
| `video_favorite.json` | 收藏库 |
| `local_resources.json` | 本地资源索引库(手动维护) |

## 注意事项

- 本工具仅供学习交流,请勿用于商业用途,尊重影视版权
- 解析接口来自第三方,不保证长期可用,失效请切换备用接口
- "VLC 播放"需要先安装 [VLC media player](https://www.videolan.org/),没装则自动 fallback 到 Chrome 或系统浏览器

## 开发说明

### 模块结构
- `find_vlc_path()` / `find_chrome_path()` — 播放器探测
- `play_url(url)` — 播放链入口(VLC → Chrome → webbrowser)
- `search_local_resources(kw, fav, hist)` — 本地搜(库 + 收藏 + 历史)
- `open_platform_search(platform, kw)` — 跳平台搜索页
- `load_local_resources()` / `save_local_resources()` — 本地库读写
- `extract_video_title(url)` — 智能 fallback 标题提取

### 为什么不做"远程真搜索"?
爱奇艺/腾讯/优酷/芒果的搜索页都是 JS 渲染的 SPA(只有空 HTML 壳子,
真实数据靠 client 端 fetch)。要绕过需要:浏览器自动化(playwright/selenium)+
几千行反爬代码 + 维护 cookie 池。这远超一个小工具的合理范围。
